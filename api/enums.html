Enums#

Type-safe Dart enums that wrap the engine’s C-side enum values.

enum ChannelType#

enum ChannelType

Speaker layout for [System.openDevice].

Values are kept in lockstep with YSE::CHANNEL_TYPE (see YseEngine/headers/enums.hpp) so the Dart enum value cast equals the underlying C enum.

Values#

auto

Pick stereo when possible.

mono

stereo

quad

surround51

5.1 surround.

surround51Side

5.1-side variant.

surround61

6.1 surround.

surround71

7.1 surround.

custom

Custom layout — the caller is expected to set speaker positions.

Properties#

The raw FFI enum value passed to the C ABI.

enum VaWaveform#

enum VaWaveform

Oscillator waveform for a synth’s virtual-analog voice ([Synth.setVaOscWave]).

Values match YSE::SYNTH::VA_WAVEFORM (see synth/vaVoice.hpp).

Values#

saw

Band-limited sawtooth.

pulse

Band-limited pulse with variable width (PWM).

triangle

Band-limited triangle.

sine

Sine.

noise

White noise.

wavetable

Morph across the wavetable bank (see [Synth.loadVaWavetable]).

enum PCategory#

enum PCategory

Documentation category a patcher object type is filed under ([PatcherObjectType.category]).

Drives the section headings on the patcher object reference. Values match YSE::PATCHER::pCategory (mirrored by the C enum YsePCategory).

Values#

unset

Uncategorised — should not reach a shipped object in practice.

oscillator

Oscillators / signal generators.

filter

Filters.

math

Arithmetic / math objects.

generic

Generic routing / utility objects.

gui

GUI control objects.

time

Timing objects.

midi

MIDI message objects.

PCategory(this.native)

Maps a raw C-side [raw.YsePCategory] to its Dart enum, defaulting to [PCategory.unset] for unknown values.

Properties#

enum InletAccepts#

enum InletAccepts

One message kind a patcher inlet accepts ([PatcherInlet.accepts]).

The engine reports the set as an OR-ed bitmask; the wrapper decodes it into a Set<InletAccepts>. Values match YSE::PATCHER::InletType (mirrored by the C enum YseInletAccepts).

Values#

buffer

Accepts an audio-rate DSP buffer.

float

Accepts a float.

integer

Accepts an integer.

bang

Accepts a bang.

list

Accepts a list.

Constructors#

InletAccepts(this.native)

Static methods#

Decodes an engine accepts bitmask into the set of flags it encodes.
