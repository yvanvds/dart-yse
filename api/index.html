API Reference

API Reference#

The public dart-yse API lives in the yse package and is imported through a single library:

import 'package:yse/yse.dart';

The pages below are generated from each class’s dartdoc comments by tool/emit_api_rst.dart (run by make api and make html). The same comments drive dart doc and pub.dev — keep them as the canonical source of truth.

Every wrapper class points back at the upstream C++ class it wraps so you can read both views of the API side by side. Look for the Upstream: line at the top of each subsystem page.
