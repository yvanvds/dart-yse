Patcher object reference#

Every patcher object that the engine’s patcher::CreateObject knows about. Each entry shows the object’s description, inlet/outlet/parameter documentation pulled directly from the C++ source via the engine’s ADD_DESCRIPTION / INLET_DOC / OUTLET_DOC / PARAM_DOC macros.

Pass the type string (or the matching yse.Obj constant) as the first argument to yse.Patcher.createObject(). The ~ prefix marks DSP / audio-rate objects; . marks control-rate objects.
