Pos#

class Pos

3D position vector used everywhere YSE needs spatial coordinates — sound positions, the listener position, reverb-zone centres.

Immutable value type. Construct fresh instances instead of mutating.

YseException#

class YseException implements Exception

Exception type for every failure surfaced from the YSE engine.

Wraps a [YseStatus] code (when the underlying call returned one) and a human-readable message — typically the engine’s thread-local last-error string at the time of failure.

Constructors#

YseException(this.message, [this.status])

Constructs an exception with a [message] and optional [status].

Properties#

The raw status code, if the originating C call returns one.
