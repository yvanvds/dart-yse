dart-yse
========

Dart bindings for the `libYSE <https://github.com/yvanvds/yse-soundengine>`_
sound engine — 3D audio playback, mixing, DSP, MIDI, and generative music.
The bindings are a thin Dart FFI layer over the engine's flat ``extern "C"``
ABI; everything you can do in the C++ API has a Dart equivalent on the
:doc:`api/index`.

New here? Start with :doc:`intro/index`, then :doc:`intro/mental_model`,
then :doc:`intro/hello_sound`. Looking for a specific class? The
:doc:`api/index` is the full reference.
