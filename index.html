dart-yse

dart-yse#

Dart bindings for the libYSE sound engine — 3D audio playback, mixing, DSP, MIDI, and generative music. The bindings are a thin Dart FFI layer over the engine’s flat extern "C" ABI; everything you can do in the C++ API has a Dart equivalent on the API Reference.

New here? Start with What is dart-yse?, then Mental model, then Hello, sound. Looking for a specific class? The API Reference is the full reference.
